# SensorDipoloMagnético
Se trata de detectar y plotear en tiempo real el campo magnético producido por un dipolo.


# Configuración del experimento
Alineamos un magnetómetro BMM150 con un sensor láser de distancia VL53L1X. El sujeto de prueba es un imán de neodimio N50 con 3cm de diámetro y 0,4 cm de grosor ubicado a 30 cm de los detectores. El punto es que vaya dando pasos "infinitesimales" hasta llegar a una distancia del sensor tal que no se sature.
